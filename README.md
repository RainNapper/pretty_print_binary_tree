# Description

        8             8888              8               8               4               4                              48
      /   \           /   \           /   \           /   \           /   \           /   \                               \
    4       4      444     444      4       4       4       4       2       2       2     -222                             12
   / \     / \     / \     / \     / \     / \     / \     / \                                                               \
  2   2   2   2  22  22  22  22   2   2   2   2   2   2   2   2                                                              -3
